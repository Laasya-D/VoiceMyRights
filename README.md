# 🎨 VoiceMyRights Frontend

Frontend for the VoiceMyRights legal agreement analyzer.

## Features

- Responsive landing page
- Legal document input
- Sample agreements
- Dynamic results dashboard
- Interactive report page
- PDF report download
- GitHub navigation
- Smooth animations

## Built With

- HTML5
- CSS3
- Vanilla JavaScript

## Pages

- index.html
- results.html
- report.html

## Run

Simply open

```
index.html
```

or deploy on Vercel.
