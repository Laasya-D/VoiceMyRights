# ⚖️ VoiceMyRights

> AI-powered legal agreement analyzer that translates complex contracts into plain English.

## 📖 Overview

Every day, users accept Terms of Service, privacy policies, employment contracts, and rental agreements without understanding the legal implications.

**VoiceMyRights** helps users make informed decisions by analyzing legal documents, identifying potentially risky clauses, and presenting easy-to-understand explanations with a Legal Risk Score.

---

## ✨ Features

- 🔍 Legal Risk Analysis
- 📊 Dynamic Risk Score (0–100)
- 🚩 Detection of high-risk clauses
- 🔐 Privacy concern identification
- 💡 Actionable recommendations
- 📄 Downloadable legal summary report
- 🎯 Sample agreements for quick testing

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/VoiceMyRights.git
```

---

### Backend

```bash
cd backend
npm install
npm start
```

Runs on:

```
http://localhost:3000
```

---

### Frontend

Open

```
frontend/index.html
```

or deploy using Vercel.
